- ## §4.1 刚体的运动及描述
- ### 一、刚体的定义与自由度
  
  **刚体（Rigid Body）：** 在任何情况下形状和体积都不改变的物体（理想化模型）。本质上是特殊的质点系，各质元间相对位置保持不变（任意两点间距离始终保持不变）。
  
  **自由度：** 完全描述运动所需的独立坐标数。
  
  | 运动类型 | 自由度 | 说明 |
  |----------|--------|------|
  | 质点的直线运动 | 1 | 一个变数即可描述 |
  | 质点的一般运动 | 3 | 三个坐标 $(x,y,z)$ |
  | 刚体（最大） | 6 | 确定三个不共线点的位置需9个变量，减去3个距离约束 = 6 |
  | 刚体平动 | 3 | 质心坐标 $(x_C, y_C, z_C)$ |
  | 刚体定轴转动 | 1 | 仅需转角 $\theta$ |
  | 刚体定点转动 | 3 | 转轴方向(2) + 绕轴转角(1) |
  | 刚体平面平行运动 | 3 | 质心平动(2) + 绕质心转动(1) |
  | 刚体一般运动 | 6 | 基点平动(3) + 绕基点定点转动(3) |
- ### 二、刚体的运动形式
  
  **1. 平动（Translation）**
- 特征：连接刚体内任意两点的直线在任意时刻的位置都彼此平行
- 刚体上所有点的运动轨迹相同
- 可用质心（或任何一点）的运动代表整体
- 自由度 = 3
  
  **2. 转动（Rotation）**
- 定轴转动：各质元均做圆周运动，所有圆心在同一条固定的直线（转轴）上。自由度 = 1。**（本章重点）**
- 定点转动：刚体上只有一点固定不动，刚体绕过该固定点的某一瞬时轴线转动。自由度 = 3（如陀螺运动）
  
  **3. 平面平行运动**
- 刚体上各点都平行于某一固定平面的运动
- 可分解为：随质心的平动（自由度2）+ 绕质心且垂直于运动平面的定轴转动（自由度1）
- 总自由度 = 3（如车轮直线滚动）
  
  **4. 一般运动**
- 刚体不受任何限制的任意运动
- 可分解为：随基点 O 的平动（自由度3）+ 绕过基点 O 的瞬时轴的定点转动（自由度3）
- 基点选取不同，平动描述不同，转动描述也可能不同
- 总自由度 = 6
- ### 三、刚体转动的运动学描述（角量体系）
-
- 角速度矢量  $\vec{\omega}$  和角加速度矢量 $\vec{\alpha}$
- 定轴转动的特点：刚体各个部分在相同时间内绕转轴转过的角度（角位移）都相同
- 因此引入角量描述非常方便：角坐标 $\theta$、角位移 $\Delta\theta$ 等
- 定轴转动刚体上任意点的 $\omega$、$\alpha$ 都相同
  
  **线量与角量的关系：**
- 线速度：$\vec{v} = \vec{\omega} \times \vec{r}$ ，大小 $v = \omega r$（$r$ 为到转轴的垂直距离）
- 切向加速度：$\vec{a}_t = \vec{\alpha} \times \vec{r}$ ，大小 $a_t = \alpha r$
- 法向加速度：$a_n = \omega^2 r$（方向指向转轴）
- 矢量形式：$\vec{a}_n = -\omega^2 \vec{r}$（或 $\vec{a}_t = \vec{\alpha} \times \vec{r}$)
  
  **匀角加速转动运动学公式**（$\alpha = \text{const}$）：
  $$\omega = \omega_0 + \alpha t$$
  $$\theta = \theta_0 + \omega_0 t + \frac{1}{2}\alpha t^2$$
  $$\omega^2 - \omega_0^2 = 2\alpha(\theta - \theta_0)$$
  
  > **考点提示：** 这三组公式与匀加速直线运动公式完全类比 — $\theta \leftrightarrow x$, $\omega \leftrightarrow v$, $\alpha \leftrightarrow a$。熟练运用此类比可以快速解题。

- ## §4.2 刚体的定轴转动
- ### 一、外力矩及对转轴的分量
- 设第 $i$ 个质元受外力 $\vec{F}_i$（假定 $\vec{F}_i$ 垂直于转轴）
- 力 $\vec{F}_i$ 对参考点 O 的力矩 $\vec{M}_O = \vec{R}_i \times \vec{F}_i$（其中 $\vec{R}_i$ 为 O 到 $\vec{F}_i$ 作用点的矢径）
- 在定轴转动中，力矩的垂直于转轴的分量不可能引起刚体绕轴转动，**被轴承提供的反作用力矩所抵消，可以丢弃**
- **关键：** 只需考虑力矩在转轴（设为 $z$ 轴）上的分量
- 力对参考点 O 的力矩在 $z$ 轴上的分量 = 力对转轴的垂足 O'（转心）的力矩 = **力对转轴的力矩**
- 合外力矩的 $z$ 分量：$M_z = \sum (\vec{r}_i \times \vec{F}_i)_z$
- ### 二、定轴转动刚体的角动量
- 第 $i$ 个质元对 O 点的角动量：$\vec{L}_i = \vec{R}_i \times (\Delta m_i \vec{v}_i)$
- 只考虑 $z$ 方向的分量：$L_z = \sum \Delta m_i r_i^2 \omega = (\sum \Delta m_i r_i^2) \omega$
- 定义**转动惯量** $J_z = \sum \Delta m_i r_i^2$（$r_i$ 为质元到转轴的垂直距离）
- 则 $L_z = J_z \omega$
- ### 三、刚体定轴转动定律（核心）
  
  $$M_z = \frac{dL_z}{dt} = \frac{d(J_z \omega)}{dt} = J_z \alpha$$
  
  定轴转动时，可简记为：$M = J\alpha$
  
  **与牛顿第二定律的类比：**
  
  | 平动 | 转动 |
  |------|------|
  | 力 $F$ | 力矩 $M$ |
  | 质量 $m$ | 转动惯量 $J$ |
  | 加速度 $a$ | 角加速度 $\alpha$ |
  | $F = ma$ | $M = J\alpha$ |
  
  > **核心理解：** $J$ 反映刚体转动的惯性 — 转动惯量越大，在相同力矩下角加速度越小。
- ### 四、刚体定轴转动的角动量定理与角动量守恒
  
  **角动量定理（冲量矩形式）：**
  $$\int_{t_0}^{t} M_z dt = J_z \omega - J_z \omega_0$$
  
  其中 $\int_{t_0}^{t} M_z dt$ 称为 $t_0$ 到 $t$ 时间内作用在刚体上的**冲量矩**。
  
  **角动量守恒定律：**
  当合外力矩 $M_z = 0$ 时，$J_z \omega = \text{const}$。
  
  角动量守恒的三种情况：
- (a) $J$、$\omega$ 都不变 → $L = \text{const}$（刚体定轴匀角速转动）
- (b) $J$、$\omega$ 都变化，但 $J\omega = \text{const}$（如花样滑冰、芭蕾舞、跳水运动员收臂/伸展改变转速）
- (c) 刚体组角动量守恒：$\sum J_i \omega_i = \text{const}$（多刚体系统，绕同一轴转动，角动量可在刚体组内部传递）

- ### 例题一（§4.2 — Atwood机含滑轮）
  
  > **题目：** 如图所示，质量分别为 $m_A$、$m_B$ 的物体 A 和 B 通过轻绳跨过半径为 $r$、转动惯量为 $J$ 的定滑轮（可绕水平固定轴转动）。设 $m_A > m_B$，绳与滑轮间无滑动。求：物体的加速度 $a$ 和绳中张力。
  
  **研究对象：** A、B、圆柱（滑轮）
  
  **受力分析与动力学方程：**
  
  对物体 A：
  $$m_Ag - T_A = m_A a$$
  
  对物体 B：
  $$T_B - m_Bg = m_B a$$
  
  对圆柱（滑轮）— 转动定律：
  $$T_A r - T_B r = J\alpha$$
  
  **附加约束方程：** $a = \alpha r$ （绳与滑轮间无滑动）
  
  **求解：** 联立以上四式解得：
  
  $$a = \frac{(m_A - m_B)g}{m_A + m_B + J/r^2}$$
  
  $$\alpha = \frac{a}{r}$$
  
  > **考点提炼：**
  > 1. 滑轮有质量时必须用转动定律，不可简单认为两边张力相等
  > 2. 约束条件 $a = \alpha r$ 是连接平动与转动的桥梁
  > 3. $J/r^2$ 可视为滑轮的"等效平动质量"

- ### 例题二（§4.2 — 含"碰撞"的滑轮系统）
  
  > **题目：** 在例题一的基础上，设物体 C（质量为 $m_C$）初始静止置于物体 B 上方（不连接）。A 和 B 从静止开始运动，当运动距离 $S$ 后，细绳突然被拉紧（"碰撞"），物体 C 被带动。求"碰撞"后物体 C 的速度 $V_C$。
  
  **已知：** 碰撞前 A、B 的速度 $v_0$（由例题一的加速度 $a$ 和运动距离 $S$ 求得：$v_0^2 = 2aS$）
  
  **研究对象：** A, B, C, 圆柱
  
  **分析方法：碰撞过程中忽略常规力（重力、摩擦力等），仅考虑冲击力（绳中张力）的冲量。**
  
  设碰撞持续时间为 $\Delta t$：
  
  对 A（由质点动量定理）：
  $$-T_A \Delta t = m_A V_A - m_A v_0$$
  
  对 B：
  $$(T_B - T_C) \Delta t = m_B V_B - m_B v_0$$
  
  对圆柱（由刚体角动量定理）：
  $$(T_A - T_B)r \Delta t = J \frac{V_A}{r} - J \frac{v_0}{r}$$
  
  对 C：
  $$T_C \Delta t = m_C V_C - 0$$
  
  **约束条件（碰撞后）：** $V_A = V_B = V_C = V$，$\omega = V/r$
  
  **求解：** 将四个方程相加（张力项相消），得：
  
  $$V \left(m_A + m_B + m_C + \frac{J}{r^2}\right) = v_0 \left(m_A + m_B + \frac{J}{r^2}\right)$$
  
  $$V = \frac{m_A + m_B + J/r^2}{m_A + m_B + m_C + J/r^2} \cdot v_0$$
  
  > **考点提炼：**
  > 1. **碰撞/冲击过程中可忽略常规力**（重力等）的冲量，只考虑冲击力的冲量 — 这是简化计算的关键
  > 2. 此式 $J = 0$ 时形式与动量守恒相似，但**绝不是简单的动量守恒**！这是因为有滑轮提供的外力矩（通过轴）
  > 3. 等效方法：若忽略常规力并**考虑对转轴的角动量守恒**，也可得到相同结果。思路：
  >    - 碰撞前角动量：$m_Av_0r + m_Bv_0r + J(v_0/r)$
  >    - 碰撞后角动量：$(m_A + m_B + m_C)Vr + J(V/r)$
  >    - 令两者相等可得相同结果

- ### 例题三（§4.2 — "打击中心"问题）
  
  > **题目：** 一细杆质量为 $m$，长度为 $l$，可绕其一端 O 处的水平光滑轴转动，初始在竖直位置静止。若在某时刻有力 $F$ 水平作用在距 O 点为 $l_0$ 的 A 处，求此时轴对杆的作用力（支反力）$\vec{F}_x$、$\vec{F}_y$。
  
  **研究对象：** 细杆
  
  **解：**
  
  除力 $F$ 外，系统还受重力、轴的支反力。但重力与支反力对轴 O 的力矩为零。
  只有 $F$ 对细杆绕 O 的转动有影响：
  $$M_O = F l_0$$
  
  细杆绕 O 的转动惯量：$J_O = \frac{1}{3}m l^2$
  
  由转动定律：
  $$F l_0 = \frac{1}{3}m l^2 \cdot \alpha \quad \Rightarrow \quad \alpha = \frac{3F l_0}{m l^2}$$
  
  质心位置（距 O）：$l_C = l/2$
  
  质心加速度的分量：
- 切向（水平）：$a_{Cx} = \alpha \cdot \frac{l}{2} = \frac{3F l_0}{2ml}$
- 法向：初始 $\omega = 0$，故 $a_{Cy} = 0$
  
  由**质心运动定理**：
  
  水平方向（$x$）：
  $$F + F_x = m a_{Cx} \quad \Rightarrow \quad F_x = m \cdot \frac{3F l_0}{2ml} - F = F\left(\frac{3l_0}{2l} - 1\right)$$
  
  竖直方向（$y$）：
  $$F_y - mg = m a_{Cy} = 0 \quad \Rightarrow \quad F_y = mg$$
  
  **讨论（三种情况）：**
  
  | 条件 | $F_x$ 符号 | 物理意义 |
  |------|-----------|----------|
  | $l_0 > \frac{2}{3}l$ | $F_x > 0$ | 轴对杆的水平力与 $F$ 同向 |
  | $l_0 = \frac{2}{3}l$ | $F_x = 0$ | 轴的水平支反力为零！**A 点即为打击中心** |
  | $l_0 < \frac{2}{3}l$ | $F_x < 0$ | 轴对杆的水平力与 $F$ 反向 |
  
  **"打击中心"定义：** 当 $l_0 = \frac{2}{3}l$ 时，轴的水平支反力 $F_x = 0$，此时力的作用点 A 称为**打击中心**。
  
  > **考点提炼：**
  > 1. **打击中心的概念：** 使轴不受水平冲击反力的力作用点位置
  > 2. 不同刚体的打击中心与形状及质量分布有关
  > 3. 实际应用：使用工具（如锤子、球棒）敲打物体时，应尽量用打击中心位置击打，以避免手感受到较大的反作用力
  > 4. 计算步骤：转动定律 → 求质心加速度 → 质心运动定理 → 求支反力

- ### 例题四（§4.2 — 转台+人，角动量守恒）
  
  > **题目：** 水平转台（质量 $m_1$，半径 $R$）可绕竖直的中心轴转动，初始角速度为 $\omega_0$。一人（质量 $m_2$）初始站在转台中心，然后相对转台以恒定速率 $u$ 沿半径向边缘走去。求经过时间 $t$ 后，转台转过了多少角度。
  
  **解：**
  
  **关键：** 人与转台组成的系统对竖直轴的合外力矩为零 → **对竖直轴的角动量守恒。**
  
  系统对竖直轴的转动惯量（任意时刻）：
  $$J(t) = \frac{1}{2}m_1 R^2 + m_2 (ut)^2$$
  
  （第一项：转台；第二项：人距轴 $ut$ 处视为质点）
  
  初始角动量 = 任意时刻角动量：
  $$\left(\frac{1}{2}m_1 R^2 + 0\right) \omega_0 = \left(\frac{1}{2}m_1 R^2 + m_2 u^2 t^2\right) \omega(t)$$
  
  $$\Rightarrow \omega(t) = \frac{\frac{1}{2}m_1 R^2}{\frac{1}{2}m_1 R^2 + m_2 u^2 t^2} \cdot \omega_0$$
  
  转台转过的角度：
  $$\theta = \int_0^t \omega(t) dt = \int_0^t \frac{\frac{1}{2}m_1 R^2 \omega_0}{\frac{1}{2}m_1 R^2 + m_2 u^2 t^2} dt$$
  
  积分结果（设 $a^2 = \frac{m_1 R^2}{2m_2 u^2}$）：
  $$\theta = \frac{m_1 R^2 \omega_0}{2m_2 u^2} \cdot \frac{1}{ut} \ln\left(1 + \frac{2m_2 u^2 t^2}{m_1 R^2}\right)^{1/2}$$
  
  或等价形式（反正切形式）。
  
  > **考点提炼：**
  > 1. **何时用角动量守恒：** 系统对转轴的合外力矩为零时
  > 2. 转动惯量随时间变化时角速度也随之变化，保持 $J\omega = \text{const}$
  > 3. 类比花样滑冰运动员：手臂收拢 → $J \downarrow$ → $\omega \uparrow$；手臂伸展 → $J \uparrow$ → $\omega \downarrow$
  
  ---
- ### 例题五（§4.2 — 摩擦离合器，角动量守恒）
  
  > **题目：** 飞轮1对轴的转动惯量为 $J_1$，以角速度 $\omega_1$ 转动；摩擦轮2对同一轴的转动惯量为 $J_2$，初始静止。现让两轮沿轴向靠近并结合在一起。求结合后两轮达到的共同角速度 $\omega$。
  
  **解：**
  
  两轮对共同转轴的合外力矩为零（轴向压力过轴），角动量守恒：
  
  $$J_1 \omega_1 + J_2 \cdot 0 = (J_1 + J_2) \omega$$
  
  $$\Rightarrow \omega = \frac{J_1 \omega_1}{J_1 + J_2}$$
  
  > **考点提炼：**
  > 1. 两轮绕同轴转动，轴向结合 → 角动量守恒
  > 2. 结合过程中摩擦力是内力，不改变系统总角动量
  > 3. 总有部分动能转化为内能（热能）：$\Delta E_k = \frac{1}{2}J_1\omega_1^2 - \frac{1}{2}(J_1+J_2)\omega^2 > 0$
  
  ---
- ### 例题六（§4.2 — 齿轮正交啮合）
  
  > **题目：** 两圆盘形齿轮半径分别为 $r_1$、$r_2$，对通过盘心垂直于盘面转轴的转动惯量分别为 $J_1$、$J_2$。开始时，1轮以 $\omega_0$ 转动，2轮静止。现两轮正交啮合。求啮合后两轮的角速度 $\omega_1$、$\omega_2$。
  
  **关键：** 与例题五不同 — 两轮绕**不同轴**转动，不能直接用角动量守恒！
  
  **解：** 对两轴分别使用角动量定理：
  
  设啮合过程中齿轮间的相互作用力为 $F$。
  
  对轮1（取竖直向上正方向）：
  $$-F r_1 \cdot \Delta t = J_1 \omega_1 - J_1 \omega_0$$
  
  对轮2（取水平向右正方向）：
  $$F r_2 \cdot \Delta t = J_2 \omega_2 - 0$$
  
  **约束关系：** 单位时间内两轮转过相同的轮缘长度（啮合处线速度相等）：
  $$r_1 \omega_1 = r_2 \omega_2$$
  
  联立求解得：
  $$\omega_1 = \frac{J_1 r_2^2}{J_1 r_2^2 + J_2 r_1^2} \cdot \omega_0$$
  $$\omega_2 = \frac{J_1 r_1 r_2}{J_1 r_2^2 + J_2 r_1^2} \cdot \omega_0$$
  
  > **考点提炼：**
  > 1. **角动量守恒的前提是合外力矩为零，对同一转轴而言！** 不同轴转动时需分别列方程
  > 2. 约束条件（啮合处线速度相等）连接两个转动
  > 3. 对比例题五（同轴 → 角动量守恒）和例题六（异轴 → 角动量不守恒！需用角动量定理分别处理）

- ## §4.3 转动惯量的计算
- ### 一、定义式与计算方法
  
  **分立结构：** $J = \sum \Delta m_i r_i^2$
  
  **连续体：** $J = \int r^2 dm$
- 体分布：$dm = \rho dV$
- 面分布：$dm = \sigma dS$
- 线分布：$dm = \lambda dl$
  
  $r$ 为质元到转轴的**垂直距离**，$J$ 取决于质量大小、质量分布以及转轴的位置。
- ### 二、常用刚体的转动惯量计算
  
  **（1）均匀圆环（质量 $m$，半径 $R$），轴过中心垂直环面：**
  $$J = \int R^2 dm = R^2 \int dm = mR^2$$
  
  （所有质元到轴的距离相等，均为 $R$）
  
  **（2）均匀圆盘（质量 $m$，半径 $R$），轴过中心垂直盘面：**
  
  取半径为 $r$、宽度为 $dr$ 的圆环，$dm = \frac{m}{\pi R^2} \cdot 2\pi r dr$
  
  $$J = \int_0^R r^2 \cdot \frac{2m}{R^2} r dr = \frac{2m}{R^2} \cdot \frac{R^4}{4} = \frac{1}{2}mR^2$$
  
  **（3）均匀实心球体（质量 $m$，半径 $R$），轴沿直径：**
  
  将球体视为一系列薄圆盘叠加。距球心 $z$ 处，厚度 $dz$ 的圆盘半径 $r = \sqrt{R^2 - z^2}$，质量 $dm = \rho \pi r^2 dz = \frac{3m}{4\pi R^3} \pi (R^2 - z^2) dz$
  
  该薄圆盘对直径的转动惯量（利用垂直轴定理）：$dJ = \frac{1}{2} r^2 dm$
  
  $$J = \int_{-R}^{R} \frac{1}{2} (R^2 - z^2) \cdot \frac{3m}{4R^3} (R^2 - z^2) dz = \frac{2}{5}mR^2$$
  
  若轴沿切线：由平行轴定理 $J = \frac{2}{5}mR^2 + mR^2 = \frac{7}{5}mR^2$
  
  **球壳：** $J = \frac{2}{3}mR^2$
  
  **（4）均匀细杆（质量 $m$，长度 $l$）：**
- 轴过中心垂直杆：取 $x$ 轴沿杆，原点在中心
  $$J_C = \int_{-l/2}^{l/2} x^2 \cdot \frac{m}{l} dx = \frac{1}{12}m l^2$$
- 轴过一端垂直杆（用平行轴定理）：
  $$J_O = J_C + m\left(\frac{l}{2}\right)^2 = \frac{1}{12}ml^2 + \frac{1}{4}ml^2 = \frac{1}{3}ml^2$$
- ### 三、平行轴定理
  
  $$J_O = J_C + md^2$$
  
  其中 $J_C$ 为对通过质心的平行轴的转动惯量，$d$ 为两平行轴之间的距离。
  
  **物理意义：** 刚体对任意轴的转动惯量 = 对通过质心的平行轴的转动惯量 + 刚体总质量 $\times$ 两轴距离平方。
  
  **应用举例 — 挂钟摆锤的转动惯量：**
  摆锤由细杆（$m_1$, $l$）和圆盘（$m_2$, $R$）组成，轴在杆顶端O。
  $$J_O = \frac{1}{3}m_1 l^2 + \left[\frac{1}{2}m_2 R^2 + m_2 (l+R)^2\right]$$
  
  （杆：$J_O = \frac{1}{3}m_1 l^2$；盘：用平行轴定理，轴距盘心 $l+R$）
- ### 四、垂直轴定理（仅适用于薄平板刚体）
  
  $$J_z = J_x + J_y$$
  
  其中 $z$ 轴垂直于薄板平面，$x$、$y$ 轴在板平面内且三者交于一点。
  
  **应用举例：** 均匀圆盘对直径的转动惯量
  
  已知 $J_z = \frac{1}{2}mR^2$（对垂直盘面的轴），由对称性 $J_x = J_y$，故：
  $$J_x = J_y = \frac{1}{2}J_z = \frac{1}{4}mR^2$$
  
  > **考点提醒：** 垂直轴定理**仅适用于薄平板刚体**（厚度可忽略），不可随意推广到三维刚体。

- ## §4.4 定轴转动中的功能关系
- ### 一、刚体定轴转动的动能
  
  $$E_k = \frac{1}{2} J \omega^2$$
  
  可利用平行轴定理分解为：
  $$E_k = \frac{1}{2} J_C \omega^2 + \frac{1}{2} m v_C^2 = \frac{1}{2} m (\omega d)^2 + \frac{1}{2} J_C \omega^2$$
  
  （质心绕定轴作圆周运动的动能 + 绕质心转动的动能 = 柯尼希定理）
- ### 二、力矩的功
  
  设外力 $F_i$ 作用在质元 $\Delta m_i$ 上（位于转动平面内），刚体转过 $d\theta$：
  
  $$dW_i = F_i \cdot r_i d\theta = M_{iz} d\theta$$
  
  总功：
  $$W = \int_{\theta_1}^{\theta_2} M_z d\theta$$
  
  （若 $M_z = \text{const}$，则 $W = M_z \Delta\theta$）
- ### 三、刚体定轴转动的动能定理
  
  由转动定律 $M_z = J \frac{d\omega}{dt}$，以及 $d\theta = \omega dt$：
  
  $$W = \int M_z d\theta = \int J \frac{d\omega}{dt} \cdot \omega dt = \int_{\omega_1}^{\omega_2} J \omega d\omega = \frac{1}{2}J\omega_2^2 - \frac{1}{2}J\omega_1^2$$
  
  **即：合外力矩的功 = 转动动能的增量。**
- ### 四、刚体的重力势能
  
  刚体与地球系统的重力势能 = 将全部质量集中在质心时的势能：
  $$E_p = mgh_C$$
  
  （以地面为零势能面，$h_C$ 为质心高度）
- ### 五、功能原理与机械能守恒
  
  **功能原理：** $W_{\text{非保}} = \Delta E_k + \Delta E_p$
  
  其中 $W_{\text{非保}}$ = 除重力以外的其它外力矩的功。
  
  **机械能守恒定律：** 若除重力外其它外力矩不做功（或无非保守力做功），则：
  
  $$\frac{1}{2}J\omega^2 + mgh_C = \text{const}$$

- ### 例题七（§4.4 — 细棒从水平释放）
  
  > **题目：** 均质细棒质量 $m$，长度 $l$，可绕水平轴 O（在棒的一端）自由转动。开始时棒处于水平位置并由静止释放。求：
  > （1）水平位置放手时，棒的质心加速度；
  > （2）摆到竖直位置时，棒的角速度；
  > （3）摆到竖直位置时，轴的支反力。
  
  **解：**
  
  **（1）水平位置时：**
  
  在水平位置，重力力矩 $M = mg \cdot \frac{l}{2}$
  
  $J_O = \frac{1}{3}ml^2$
  
  由转动定律：$mg \cdot \frac{l}{2} = \frac{1}{3}ml^2 \cdot \alpha$
  
  $$\Rightarrow \alpha = \frac{3g}{2l}$$
  
  质心加速度（质心在 $l/2$ 处）：
- 切向：$a_{Ct} = \alpha \cdot \frac{l}{2} = \frac{3g}{4}$
- 法向：初始 $\omega = 0$，$a_{Cn} = 0$
  
  **（2）从水平到竖直，由机械能守恒：**
  
  选 O 点为势能零点。
  
  水平位置：$E_1 = 0 + mg\left(-\frac{l}{2}\right) = -\frac{1}{2}mgl$
  
  竖直位置：$E_2 = \frac{1}{2}J_O \omega^2 + mg\left(-l\right) = \frac{1}{2} \cdot \frac{1}{3}ml^2 \cdot \omega^2 - mgl$
  
  由 $E_1 = E_2$：
  $$-\frac{1}{2}mgl = \frac{1}{6}ml^2 \omega^2 - mgl$$
  
  $$\Rightarrow \omega^2 = \frac{3g}{l}, \quad \omega = \sqrt{\frac{3g}{l}}$$
  
  **（3）竖直位置时，应用质心运动定律求支反力：**
  
  质心加速度：
- 切向：$\alpha = 0$（重力过轴，力矩为0），$a_{Ct} = 0$
- 法向（向上为正）：$a_{Cn} = \omega^2 \cdot \frac{l}{2} = \frac{3g}{l} \cdot \frac{l}{2} = \frac{3g}{2}$
  
  由质心运动定律（竖直方向）：
  $$F_y - mg = m a_{Cn} = m \cdot \frac{3g}{2}$$
  
  $$\Rightarrow F_y = mg + \frac{3}{2}mg = \frac{5}{2}mg$$
  
  > **考点提炼：**
  > 1. 含转动的机械能守恒：势能零点可以灵活选取（这里选 O 点简化计算）
  > 2. 求支反力的标准步骤：转动定律 → 角加速度 → 质心加速度 → 质心运动定理 → 支反力
  > 3. 竖直位置时 $\alpha = 0$（重力过轴），但 $\omega$ 最大 → 法向加速度最大 → 支反力最大（$2.5mg$ > $mg$）

- ### 例题八（§4.4 — 圆锥体+滑块，角动量+机械能守恒）
  
  > **题目：** 圆锥体半径为 $R$，高为 $h$，转动惯量为 $J$，表面有浅槽。令其以 $\omega_0$ 绕竖直对称轴转动，小滑块 $m$ 由静止从顶端开始沿槽下滑（不计摩擦）。求滑块滑到底部时滑块的速度 $u$ 和圆锥体的角速度 $\omega$。
  
  **解：**
  
  系统：圆锥体 + 滑块 + 地球
  
  **（1）对竖直轴的角动量守恒：**
  
  初始：$L_0 = J \omega_0 + 0$
  
  末态（滑块在底部距轴 $R$）：$L = J \omega + m R^2 \omega$
  
  $$J \omega_0 = (J + m R^2) \omega \quad \Rightarrow \quad \omega = \frac{J}{J + mR^2} \omega_0$$
  
  **（2）系统机械能守恒（不计摩擦）：**
  
  初始：$E_0 = \frac{1}{2}J\omega_0^2 + mgh + 0$
  
  末态：$E = \frac{1}{2}J\omega^2 + 0 + \frac{1}{2}m u^2$
  
  注意：滑块速度 $u$ 是相对地面的绝对速度。由于滑块在圆锥表面运动，$u$ 包含沿槽下滑分量和随圆锥转动的切向分量。但若槽为浅槽，滑块切向速度 $= \omega R$，径向和竖直分量合成 $u_r$，则 $u^2 = u_r^2 + (\omega R)^2$。
  
  由机械能守恒：
  $$\frac{1}{2}J\omega_0^2 + mgh = \frac{1}{2}J\omega^2 + \frac{1}{2}m u^2$$
  
  代入 $\omega$ 即得 $u$。
  
  > **考点提炼：**
  > 1. 同时使用**角动量守恒**和**机械能守恒**是此类组合问题的标准方法
  > 2. 注意滑块速度是绝对速度（相对于地面），不是相对圆锥体的速度
  > 3. 角动量守恒用于不能直接用轴方向力矩判断的情况

- ## §4.5 刚体进动
- ### 一、进动现象
  
  **陀螺的运动：**
- 情况一（竖直）：高速旋转的陀螺可以"立而不倒"
- 情况二（斜放）：
	- 若不高速自转 → 因重力力矩作用而倾倒
	- 若高速自转 → 自转轴绕竖直轴旋转 → **进动（回转现象）**
	  
	  **进动的物理本质：**
	  
	  对于质量分布具有轴对称性的刚体，其对转轴上任意点的角动量与角速度平行：
	  $$\vec{L} = J \vec{\omega}$$
	  
	  当重力对支点 O 产生力矩 $\vec{M}$ 时，$\vec{M}$ 的方向始终垂直于 $\vec{L}$（即垂直于自转轴与竖直轴构成的平面）。
	  
	  由角动量定理：$d\vec{L} = \vec{M} dt$
- $\vec{M} \perp \vec{L}$ → $d\vec{L} \perp \vec{L}$ → $\vec{L}$ 只改变方向、不改变大小
- 自转轴绕竖直轴旋转 → 产生进动
- ### 二、进动角速度
  
  $$\omega_p = \frac{M}{J\omega}$$
  
  其中 $M$ 为外力矩大小（通常为重力矩 $M = mgl\sin\theta$），$\omega$ 为自转角速度，$J$ 为对自转轴的转动惯量。
  
  **方向判断：** 满足 $\vec{M} = \vec{\omega}_p \times \vec{L}$
  
  **注意：** 当自转角速度方向相反时，角动量反向，进动方向也随之反向！
- ### 三、回转仪（杠杆式）
- 平衡时：$\vec{\omega}$ 大小方向不变
- 重物移近 → 受外力矩 → 出现回转现象（进动）
- ### 四、回转效应的应用
- 枪膛、炮膛中的来复线（使弹头高速自转，保持飞行稳定）
- 飞机、轮船、导弹中的惯性制导（陀螺仪）
- 自行车行驶中的稳定性（车轮旋转的陀螺效应）
  
  > **考点提醒：**
  > 1. 进动不是因为"重力拉倒了陀螺"，恰恰相反，是重力矩导致的角动量矢量旋转
  > 2. 进动角速度 $\omega_p \propto 1/\omega$ — 自转越快，进动越慢
  > 3. 公式 $\omega_p = M/(J\omega)$ 要求质量分布轴对称

- ## §4.6 刚体的平面平行运动
- ### 一、基本描述
  
  平面平行运动 = 质心平动 + 绕过质心且垂直于运动平面的轴的定轴转动。
  
  **动力学方程：**
  
  （1）质心平动（质心运动定理）：
  $$\sum \vec{F}_i = m \vec{a}_C$$
  
  （2）绕质心转动：
  $$M_C = J_C \alpha$$
  
  这是在**质心系**中的转动定律。注意：在非惯性系（质心系）中，惯性力对质心的力矩为零（因为惯性力作用在各质元上的合力通过质心），因此绕质心的转动定律与惯性系中形式完全相同！
- ### 二、刚体平面运动的动能（柯尼希定理）
  
  $$E_k = \frac{1}{2} m v_C^2 + \frac{1}{2} J_C \omega^2$$
  
  （平动动能 + 绕质心转动动能）
- ### 三、纯滚动约束条件
  
  $$v_C = \omega R \quad \text{（或 } a_C = \alpha R \text{）}$$
  
  纯滚动时接触点相对地面瞬时静止（速度为零）。

- ### 例题九（§4.6 例1 — 力作用在自由杆上）
  
  > **题目：** 一均匀细杆质量为 $m$，长度 $l$，静置于光滑水平面上。在杆的一端沿垂直于杆的方向施加一力 $F$，力的作用点到质心的距离为 $d$。求质心加速度和杆的角加速度。
  
  **解：**
  
  杆对过质心垂直杆的轴转动惯量：$J_C = \frac{1}{12}ml^2$
  
  质心运动定理：$F = m a_C \Rightarrow a_C = \frac{F}{m}$
  
  绕质心转动定律：$Fd = J_C \alpha \Rightarrow \alpha = \frac{Fd}{\frac{1}{12}ml^2} = \frac{12Fd}{ml^2}$
  
  > **考点提炼：**
  > - 平面运动分解为平动+转动的标准解题步骤
  > - 力作用线不过质心时会同时产生平动和转动加速度

- ### 例题十（§4.6 例1** — 冲量打击自由杆）
  
  > **题目：** 在例题九中，如果通过打击方式给杆一个冲量 $I$（作用时间极短），结果如何？
  
  **解：**
  
  由动量定理（平动）：$I = m v_C \Rightarrow v_C = \frac{I}{m}$
  
  由角动量定理（对质心）：$I d = J_C \omega \Rightarrow \omega = \frac{I d}{\frac{1}{12}ml^2} = \frac{12 I d}{m l^2}$
  
  > **注意：** 如果打击点恰好在端点（$d = l/2$），则 $v_C = I/m$，$\omega = 6I/(ml)$。打击后杆上各点的速度 = 质心速度 + 相对质心的转动线速度。

- ### 例题十一（§4.6 例2 — 圆柱绕绳下落）
  
  > **题目：** 一均匀圆柱体质量为 $m$，半径为 $R$，绕其对称轴的转动惯量为 $J_C = \frac{1}{2}mR^2$。在其中部绕一根细绳，绳的一端固定在天花板上。设圆柱体初速度为零，求当圆柱体从静止开始下落高度 $h$ 时，圆柱体轴心的速度 $v_C$ 和绳子拉力 $T$。
  
  **解：**
  
  **约束条件（绳不伸长、不滑动）：** 圆柱边缘线速度 = 轴心下落速度
  $$v_C = \omega R$$
  （同理：$a_C = \alpha R$）
  
  **质心运动定理（向下为正）：**
  $$mg - T = m a_C$$
  
  **绕质心转动定律：**
  $$T R = J_C \alpha = \frac{1}{2}mR^2 \cdot \alpha$$
  
  由约束条件 $\alpha = a_C / R$ 代入转动方程：
  $$T R = \frac{1}{2}mR^2 \cdot \frac{a_C}{R} \quad \Rightarrow \quad T = \frac{1}{2}m a_C$$
  
  代入平动方程：
  $$mg - \frac{1}{2}m a_C = m a_C \quad \Rightarrow \quad a_C = \frac{2}{3}g$$
  $$T = \frac{1}{2}m \cdot \frac{2}{3}g = \frac{1}{3}mg$$
  
  下落高度 $h$ 时：$v_C^2 = 2 a_C h = \frac{4}{3}gh$，即 $v_C = \sqrt{\frac{4}{3}gh}$
  
  > **考点提炼：**
  > 1. 绳子绕在圆柱上 → 圆柱下落的同时旋转 → 约束 $v_C = \omega R$
  > 2. 张力同时影响平动和转动 → 联立平动+转动方程求解
  > 3. $a_C < g$，因为部分重力势能转化为转动动能

- ### 例题十二（§4.6 例3 — 圆球纯滚动下斜面）
  
  > **题目：** 一半径为 $r$ 质量为 $m$ 的匀质圆球（$J_C = \frac{2}{5}mr^2$），从静止开始沿一粗糙斜面纯滚动而下。斜面倾角为 $\theta$，球从上端滚到下端球心高度差为 $h$。计算小球滚到下端时质心的速度 $v_C$ 和转动角速度 $\omega$。
  
  **解法一：动力学方法**
  
  **质心运动定理（沿斜面方向，向下为正）：**
  $$mg\sin\theta - f = m a_C$$
  
  **绕质心转动定律：**
  $$f r = J_C \alpha = \frac{2}{5}mr^2 \cdot \alpha$$
  
  **纯滚动约束：** $a_C = \alpha r$
  
  由约束代入转动方程：$f r = \frac{2}{5}mr^2 \cdot \frac{a_C}{r} \Rightarrow f = \frac{2}{5}m a_C$
  
  代入平动方程：$mg\sin\theta - \frac{2}{5}m a_C = m a_C$
  $$\Rightarrow a_C = \frac{5}{7}g\sin\theta$$
  
  （注意：若无转动（$J_C = 0$），则 $a_C = g\sin\theta$；有转动时加速度更小）
  
  运动距离 $s = h/\sin\theta$，$v_C^2 = 2 a_C s$，解得：
  $$v_C = \sqrt{\frac{10}{7}gh}, \quad \omega = \frac{v_C}{r} = \frac{1}{r}\sqrt{\frac{10}{7}gh}$$
  
  **解法二：机械能守恒**
  
  初态（顶端静止）：$E_1 = mgh$
  
  末态（底端）：$E_2 = \frac{1}{2}m v_C^2 + \frac{1}{2}J_C \omega^2$
  
  代入 $J_C = \frac{2}{5}mr^2$ 和约束 $v_C = \omega r$：
  $$mgh = \frac{1}{2}m v_C^2 + \frac{1}{2} \cdot \frac{2}{5}mr^2 \cdot \left(\frac{v_C}{r}\right)^2 = \frac{1}{2}m v_C^2 + \frac{1}{5}m v_C^2 = \frac{7}{10}m v_C^2$$
  $$\Rightarrow v_C = \sqrt{\frac{10}{7}gh}$$
  
  （与动力学方法结果一致！）
  
  **讨论 — 纯滚动对摩擦的要求：**
  
  纯滚动所需的静摩擦力：$f = \frac{2}{5}m a_C = \frac{2}{5}m \cdot \frac{5}{7}g\sin\theta = \frac{2}{7}mg\sin\theta$
  
  要保证不滑动，需要 $f \leq \mu_s N = \mu_s mg\cos\theta$：
  $$\frac{2}{7}mg\sin\theta \leq \mu_s mg\cos\theta \quad \Rightarrow \quad \mu_s \geq \frac{2}{7}\tan\theta$$
  
  若斜面不够粗糙（$\mu_s < \frac{2}{7}\tan\theta$），则球会连滚带滑 → 见例题十三。
  
  > **考点提炼：**
  > 1. 纯滚动问题两种解法（动力学 vs 机械能守恒），后者往往更简洁
  > 2. $a_C < g\sin\theta$：部分重力势能转化为转动动能 → 质心加速度减小
  > 3. 不同刚体的 $a_C$ 取决于 $J_C/(mR^2)$ 比值
  > 4. 静摩擦力方向判断：纯滚动时静摩擦力方向沿斜面**向上**（阻碍下滑趋势）
  
  ---
- ### 例题十三（§4.6 例3*** — 圆球连滚带滑下斜面）
  
  > **题目：** 与例题十二条件相同，但斜面不够粗糙，球在下滚过程中伴有滑动（连滚带滑）。求球滚到下端时质心速度 $v_C$ 和角速度 $\omega$。
  
  **关键区别：** 此时纯滚动约束 $v_C = \omega r$ **不成立**！摩擦为**滑动摩擦**（$f = \mu_k N$），方向和大小已知。
  
  **解：**
  
  $f = \mu_k mg\cos\theta$ （方向沿斜面向上）
  
  质心运动定理：$mg\sin\theta - \mu_k mg\cos\theta = m a_C$
  $$\Rightarrow a_C = g(\sin\theta - \mu_k \cos\theta)$$
  
  绕质心转动定律：$f r = \mu_k mg\cos\theta \cdot r = \frac{2}{5}mr^2 \cdot \alpha$
  $$\Rightarrow \alpha = \frac{5\mu_k g\cos\theta}{2r}$$
  
  （注意：此时 $a_C$ 和 $\alpha$ 不再由纯滚动条件关联）
  
  $s = h/\sin\theta$，由匀加速运动：
  $$v_C = \sqrt{2 a_C s} = \sqrt{2g(\sin\theta - \mu_k \cos\theta) \cdot \frac{h}{\sin\theta}}$$
  $$\omega = \sqrt{2 \alpha \cdot \frac{\omega^2}{2\alpha}} \quad \text{（由 } \theta_{\text{转角}} = s/r \text{）}$$
  $$\omega = \sqrt{2\alpha \cdot \frac{s}{r}} = \sqrt{2 \cdot \frac{5\mu_k g\cos\theta}{2r} \cdot \frac{h}{r\sin\theta}} = \sqrt{\frac{5\mu_k g h \cos\theta}{r^2 \sin\theta}}$$
  
  > **考点提炼：**
  > 1. "连滚带滑" ≠ 纯滚动 → 约束 $v_C = \omega r$ 不成立
  > 2. 此时摩擦力为滑动摩擦 $f = \mu_k N$（已知），而非未知的静摩擦力
  > 3. $a_C$ 与 $\alpha$ 独立求解，不再通过约束条件关联

- ### 例题十四（§4.6 例4 — 拉线轴问题）
  
  > **题目：** 线轴的两端是半径为 $R$ 的圆柱体，中间是半径为 $r$ 的同轴圆柱体。线轴质量为 $m$，绕对称轴的转动惯量为 $J_C$。在半径为 $r$ 的中轴上绕有细线，用与水平方向成 $\theta$ 角（$0 < \theta < \pi/2$）的拉力 $F$ 拉线。线轴置于粗糙水平面上。若为纯滚动，求线轴质心加速度 $a_C$ 和地面摩擦力 $f$。
  
  **解：**
  
  设摩擦力 $f$ 方向沿水平方向（与 $F$ 水平分力方向相反）。
  
  **质心运动定理：**
  
  水平方向：$F\cos\theta - f = m a_C$
  
  竖直方向：$N + F\sin\theta - mg = 0$
  
  **绕质心转动定律：** 拉力 $F$ 对轴心的力矩为 $F r$（中轴半径 $r$），摩擦力 $f$ 对轴心的力矩为 $f R$（外圆柱半径 $R$，摩擦力作用于接触线处）。注意两者对转动的影响方向：
  
  $$F r - f R = J_C \alpha$$
  
  **纯滚动约束：** $a_C = \alpha R$
  
  联立求解（三方程 + 一约束）即可得 $a_C$ 和 $f$。
  
  **讨论：** 当 $\theta$ 取不同值时，$a_C$ 方向可能不同，摩擦力的方向也会相应改变。存在某一临界角使得 $f = 0$。
  
  > **考点提炼：**
  > 1. 拉线轴的特色：拉力 $F$ 作用于半径 $r$（较小），摩擦力 $f$ 作用于半径 $R$（较大）→ 两个力矩竞争
  > 2. 摩擦力方向不一定与直觉一致，需要由方程求解判断
  > 3. 纯滚动问题的标准方程组：质心平动(1-2个方程) + 绕质心转动(1个方程) + 约束方程(1个)

- ### 例题十五（§4.6 例5 — 水平力作用下的圆柱纯滚动）
  
  > **题目：** 一质量为 $m$，半径为 $R$ 的均质圆柱（$J_C = \frac{1}{2}mR^2$），在水平外力 $F$ 作用下，在粗糙的水平面上作纯滚动。力的作用线与圆柱中心轴线的垂直距离为 $l$。求质心加速度 $a_C$ 和圆柱所受的静摩擦力 $f$。
  
  **解：**
  
  设静摩擦力 $f$ 方向（假设向左）。
  
  **质心运动定理（水平方向）：** $F - f = m a_C$
  
  **绕质心转动定律：** $F l - f R = J_C \alpha = \frac{1}{2}mR^2 \alpha$
  
  **纯滚动约束：** $a_C = \alpha R$
  
  联立解得：
  
  $$\alpha = \frac{2F(R + l)}{3mR^2}, \quad a_C = \frac{2F(R + l)}{3mR}, \quad f = \frac{2l - R}{3R}F$$
  
  **讨论（三种情况）：**
  
  | 条件 | $f$ 方向 | 物理意义 |
  |------|---------|----------|
  | $l > R/2$ | $f > 0$（与假设同向，即与 $F$ 反向） | 摩擦力阻碍平动，促进转动 |
  | $l = R/2$ | $f = 0$ | 无需摩擦力即可维持纯滚动！ |
  | $l < R/2$ | $f < 0$（与假设反向，即与 $F$ 同向） | 摩擦力促进平动，阻碍转动 |
  
  **若力作用点下移至轴线以下：**
  
  此时力 $F$ 的方向可能改变对转动力矩的贡献符号，需重新列转动方程。设 $F$ 的作用点在轴下方距轴 $l$ 处（力方向仍为水平）：
  
  质心平动：$F - f = m a_C$ （或 $F + f$ 取决于摩擦力方向假设）
  
  转动：$F l + f R = J_C \alpha$（注意符号变化）
  
  > **考点提炼：**
  > 1. **关键结论：** 当 $l = R/2$ 时 $f = 0$ — 即存在一个特殊作用线位置使得纯滚动不需要摩擦力！
  > 2. 此结果与打击问题中的"打击中心"概念完全类似
  > 3. 静摩擦力的方向不一定"阻碍运动" — 实际上在纯滚动中，静摩擦力可能促进平动（$l < R/2$ 时）

- ### 例题十六（§4.6 例6 — 小球在平板上受冲量后的运动）
  
  > **题目：** 质量 $m$、半径 $R$ 的匀质实心小球（$J_C = \frac{2}{5}mR^2$），静止于质量为 $M$、足够长的平板上。小球与平板间有摩擦，平板置于光滑水平面上。将一水平方向的冲量 $I$ 作用在小球上，作用点距平板表面的高度为 $h$（$0 < h < 2R$），作用线位于过球心而平行于纸面的平面内。试求小球最终作纯滚动时的角速度大小，并指明平板和小球的运动方向以及小球的角速度方向。
  
  **解：**
  
  **步骤1：冲量作用瞬间（碰撞过程，摩擦力冲量可忽略）**
  
  冲量 $I$ 作用 → 小球获得初始平动速度和角速度：
  
  由动量定理：$I = m v_{C0} \Rightarrow v_{C0} = I/m$
  
  由对质心的角动量定理（冲量作用点在球心上方 $h-R$ 处）：
  $$I(h - R) = J_C \omega_0 = \frac{2}{5}mR^2 \cdot \omega_0$$
  $$\Rightarrow \omega_0 = \frac{5I(h - R)}{2mR^2}$$
  
  初始时刻平板静止：$V_0 = 0$
  
  **步骤2：后续过程（摩擦使小球与平板最终达到纯滚动）**
  
  之后小球与平板间有滑动摩擦 → 摩擦力的冲量改变两者的运动状态。
  
  需分情况讨论（取决于 $h$ 与 $R$ 的关系）：
  
  **（a）$h > R$ 的情况（冲量作用于球心上方）**
  
  初始 $v_{C0} > 0$（小球质心向右），$\omega_0 > 0$（角速度垂直于纸面向里）。
  
  接触点速度：$v_{C0} - \omega_0 R = \frac{I}{m} - \frac{5I(h-R)}{2mR} = \frac{I}{m}\left(1 - \frac{5(h-R)}{2R}\right)$
- 若 $v_{C0} > \omega_0 R$（即 $h < \frac{7}{5}R$）→ 接触点向右滑动 → 摩擦力 $f$ 向左
	- 小球：平动减速，转动加速
	- 平板：受到向右的摩擦力 → 平板向右运动
- 若 $v_{C0} < \omega_0 R$（即 $h > \frac{7}{5}R$）→ 接触点向左滑动 → 摩擦力 $f$ 向右
	- 小球：平动加速，转动减速
	  
	  设最终达到纯滚动时，小球质心速度为 $v_C$，角速度为 $\omega$，平板速度为 $V$。
	  
	  纯滚动条件（相对平板）：$v_C - \omega R = V$
	  
	  对系统（小球+平板）水平方向无外力 → **动量守恒**：
	  $$m v_C + M V = I$$
	  
	  对小球（对质心的角动量定理）：$-fR \Delta t = J_C(\omega - \omega_0)$
	  （注意摩擦力矩的符号取决于摩擦力方向）
	  
	  对平板（动量定理）：$f \Delta t = M V$（或 $-M V$，取决于方向）
	  
	  联立可解得最终稳态。
	  
	  **关键结果（视 $h$ 与 $R$ 关系不同而不同）：**
	  
	  对应 $h > R$ 的情况：
- 小球向右运动，平板向左运动，角速度方向垂直于纸面向里（或外，取决于具体参数）
  
  对应 $h < R$ 的情况（冲量作用点在球心下方）：
- 初始 $\omega_0 < 0$（角速度反向），情况更为复杂
- 需分析 $\Delta t_1$（质心速度减至平板速度需时）与 $\Delta t_2$（角速度减至零需时）的关系
	- 若 $\Delta t_1 \geq \Delta t_2$（即 $h > \frac{M}{M+m}R$）：质心速度尚未与平板同步时，角速度已先反向 → 最终保持质心运动方向，转动反向
	- 若 $\Delta t_1 < \Delta t_2$（即 $h < \frac{M}{M+m}R$）：质心速度先与平板同步，角速度尚未反向 → 最终保持转动方向，质心运动反向
	  
	  **最终纯滚动时角速度的统一表达式：**
	  $$\omega_{\text{final}} = \frac{5[I M + m(h-R)]}{(7M+2m)mR^2} \quad \text{（具体符号取决于 } h \text{ 的范围）}$$
	  
	  > **考点提炼：**
	  > 1. **复杂系统的条理化分析：** 先分析冲量瞬间（碰撞近似），再分析摩擦驱动的过渡过程
	  > 2. 动量守恒 + 角动量定理 + 终态纯滚动条件 → 三方程联立
	  > 3. 分析接触点速度方向 → 判断摩擦力方向 → 这是解题的起点
	  > 4. $h = R$（冲量过球心）时 $\omega_0 = 0$，简化为纯平动+摩擦加速转动的经典问题
	  
	  ---

- ### 例题十八（§4.6 例8 — 滚动球接触粗糙表面后的运动）
  
  > **题目：** 一滚动球（有初始平动速度 $v_0$ 和初始角速度 $\omega_0$）接触粗糙表面。分析其后运动，判断何时达到纯滚动。
  
  **关键：** 此问题应考虑为**既滑且滚的一般情形**。
  
  初始阶段：球与表面间有滑动 → 滑动摩擦力作用
- 摩擦力改变平动速度
- 摩擦力矩改变转动角速度
  
  设初始时刻接触点速度 $v_{\text{接触}} = v_0 - \omega_0 R$
  
  若 $v_{\text{接触}} > 0$（接触点向前滑），摩擦力向后：
- $v_C$ 减小，$\omega$ 增大
- 直到 $v_C = \omega R$ → 达到纯滚动
  
  若 $v_{\text{接触}} < 0$（接触点向后滑），摩擦力向前：
- $v_C$ 增大，$\omega$ 减小（或甚至反向）
- 直到 $v_C = \omega R$ → 达到纯滚动
  
  **纯滚动开始条件：**
  $$\frac{v_C}{\omega R} = 1$$
  
  由动力学方程可求得纯滚动开始时刻的 $v_C$ 满足：
  $$v_C = \frac{v_0}{1 + \frac{J_C}{mR^2}}$$
  
  对于匀质实心球（$J_C = \frac{2}{5}mR^2$）：纯滚动时 $v_C = \frac{5}{7}v_0$。
  
  > **考点提炼：**
  > 1. **一般分析步骤：** 计算初始接触点速度 → 判断摩擦力方向 → 列平动+转动方程 → 求纯滚动条件
  > 2. 摩擦力使接触点速度趋于零，最终达到纯滚动
  > 3. 纯滚动一旦达到，若无其它变化（如斜面变陡等），将一直维持（静摩擦力仅维持纯滚动，不做功）

- ## 常见易错点与注意事项
- ### 1. 转动定律的使用条件
- **$M = J\alpha$ 必须是对同一转轴** — 力矩、转动惯量、角加速度要对应同一轴
- 在定轴转动中，**只考虑力矩在转轴上的分量** — 垂直分量被轴承抵消
- 非惯性系中使用对质心的转动定律时，**惯性力对质心的合力矩为零**（原因：惯性力 $-\Delta m_i \vec{a}_C$ 作用在各质元，合力通过质心）
- ### 2. 角动量守恒的条件
- **必须是对同一转轴的合外力矩为零**
- 不同轴转动时（如齿轮啮合），角动量不守恒！需要分别对每个轴使用角动量定理
- "碰撞"问题中，**常规力（重力等）的冲量矩可忽略**，但外力（如轴对滑轮的支反力）的力矩不可忽略
- ### 3. 纯滚动的静摩擦力
- **静摩擦力方向不一定与运动方向相反** — 取决于力矩的竞争关系
- **纯滚动中静摩擦力不做功**（接触点瞬时速度为零）
- 纯滚动的存在需要满足 $f \leq \mu_s N$，否则会打滑
- ### 4. 转动惯量计算
- $r$ 是质元到**转轴的垂直距离**，不是到参考点的距离
- **垂直轴定理仅适用于薄平板刚体**
- 平行轴定理中的 $d$ 是**两平行轴之间的垂直距离**
- ### 5. 动量守恒 vs 角动量守恒
- 含滑轮系统 ≠ 简单动量守恒！用 $J=0$ 代回结果看似动量守恒，但实际上是角动量守恒的特例，**不可混为一谈**
- 对于含固定转轴的系统，水平方向动量一般不守恒（轴有支反力）
- ### 6. 机械能守恒
- 刚体的重力势能 = $mgh_C$（**质心高度**），不是几何中心高度
- 纯滚动中虽然有静摩擦力，但**静摩擦力不做功** → 机械能仍然守恒
- 非纯滚动（有滑动）时，摩擦力做功 → 机械能不守恒
- ### 7. 进动
- 进动角速度 $\omega_p \propto 1/\omega$ → **自转越快，进动越慢**
- 自转方向相反时，进动方向也反向
- 进动公式要求**质量分布轴对称**
- ### 8. 约束条件的建立
- 绳与滑轮间无滑动：$a = \alpha r$
- 纯滚动：$v_C = \omega R$，$a_C = \alpha R$
- 啮合齿轮：$r_1 \omega_1 = r_2 \omega_2$
- **约束条件是连接平动与转动的关键桥梁**
- ### 9. 符号与方向
- 涉及多个物体时，**务必事先约定正方向**（平动正方向、转动正方向）
- 力矩符号：使刚体沿正方向转动的力矩为正
- 质心运动定理中各力分量的符号需与正方向一致
- ### 10. 常见数值错误
- 细杆绕端点：$J = \frac{1}{3}ml^2$（不是 $\frac{1}{12}ml^2$）
- 实心球绕直径：$J = \frac{2}{5}mR^2$（不是 $\frac{2}{3}mR^2$，那是球壳的）
- 圆柱绕中心轴：$J = \frac{1}{2}mR^2$（与圆盘相同）
- 平行轴定理：$+md^2$（加号，不是减号）
  
  ---
  
  > **全书核心：** 刚体力学 = **质心平动（牛顿定律）** + **绕质心转动（转动定律）** + **约束条件（连接平动与转动）**。掌握这一"三位一体"的分析框架，即可应对所有刚体力学问题。
