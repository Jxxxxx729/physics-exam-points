# 第一部分：第8章 平衡态

---

## 一、核心公式/概念

### 1. 基本概念

| 概念 | 定义 |
|------|------|
| **热力学系统** | 包含极大量分子（NA = 6.02×10²³）的研究对象 |
| **外界** | 热力学系统以外的物体 |
| **宏观量** | 广延量（M, V, E可累加），强度量（P, T不可累加） |
| **微观量** | 分子的 m, d, v, p, ε 等 |
| **平衡态** | 在不受外界影响条件下，系统的宏观性质不随时间改变的状态 |
| **动态平衡** | 处于平衡态时分子仍在运动碰撞，但宏观量不随时间改变 |
| **涨落** | 宏观量围绕平均值的微小起伏，分子数越多涨落越小 |
| **准静态过程** | 过程进行"无限缓慢"，每一步都可近似看作平衡态 |
| **热力学第零定律** | 若A和B、B和C分别热平衡，则A和C一定热平衡 |

### 2. 状态方程

- **理想气体状态方程（1 mol）：** PV = RT，R = 8.31 J/(mol·K)
- **质量为 m 的理想气体：** PV = (m/M)RT = νRT
- **用分子数表示：** P = nkT，其中 k = R/NA = 1.38×10⁻²³ J/K（玻耳兹曼常数）

### 3. 理想气体微观模型
- 分子当作质点，不占体积（分子线度 << 分子间平均距离）
- 分子间除碰撞瞬间外无相互作用力（忽略重力）
- 遵从牛顿力学，弹性碰撞（动能不变）

### 4. 统计假设
- 平衡态下分子速度按方向的分布各向同性
- 平衡态下分子按位置分布均匀，分子数密度处处相等
- vx平均 = vy平均 = vz平均 = 0
- vx²平均 = vy²平均 = vz²平均 = (1/3)v²平均

**统计规律特点：** 只对大量偶然事件才有意义；个别事件有偶然性但统计规律是必然的；总是伴随着涨落。

### 5. 麦克斯韦速率分布律

**速率分布函数：**
$$f(v) = 4\pi \left(\frac{m}{2\pi kT}\right)^{3/2} v^2 e^{-\frac{mv^2}{2kT}}$$

- f(v) 物理意义：在 v 附近单位速率区间内的分子数占总分子数的百分比（几率密度）
- dN = N f(v) dv

**归一化条件：** ∫₀^∞ f(v) dv = 1

**三个特征速率：**

| 名称 | 公式 | 物理意义 |
|------|------|----------|
| 最概然速率 vp | √(2kT/m) = √(2RT/M) | f(v)极大值对应速率 |
| 平均速率 v̄ | √(8kT/πm) = √(8RT/πM) | 用于碰撞、扩散、热传导 |
| 方均根速率 √(v²) | √(3kT/m) = √(3RT/M) | 反映热运动剧烈程度 |

- 温度升高 → 速率大的分子数增多，vp 右移
- 数值关系：vp < v̄ < √(v²平均)

### 6. 麦克斯韦速度分布律

**速度分量分布函数：**
$$f(v_x) = \left(\frac{m}{2\pi kT}\right)^{1/2} e^{-\frac{mv_x^2}{2kT}}$$

- 速度分布函数对速度空间原点是球对称的
- F(vx,vy,vz) = f(vx)f(vy)f(vz)
- 从速度分布到速率分布：f(v) = 4πv² F(vx,vy,vz)

**沿 x 正方向运动的分子数：** N₊ = N/2

### 7. 压强推导

- 单位时间内打到单位面积器壁的分子数：Γ = (1/4)nv̄
- **压强：** P = nkT

### 8. 温度与分子平均平动动能

$$\bar{\varepsilon}_t = \frac{3}{2} kT$$

- 平均平动动能只与温度有关
- 温度是统计概念，只能用于大量分子
- 相同温度下任意气体的平均平动动能相同
- **方均根速率因摩尔质量不同而不同：** √(v²平均) = √(3RT/M)

**0°C 时数值：** H₂ 方均根速率 ≈ 1836 m/s，O₂ 方均根速率 ≈ 461 m/s

### 9. 玻耳兹曼分布律

**重力场中分子数密度分布：**
$$n = n_0 e^{-\frac{mgz}{kT}}$$

**恒温气压公式（高度计原理）：**
$$p = p_0 e^{-\frac{mgz}{kT}} \quad \Rightarrow \quad z = \frac{kT}{mg} \ln\frac{p_0}{p}$$

**一般势场中 Boltzmann 分布：**
$$n(x,y,z) = n_0 e^{-\frac{\varepsilon_p(x,y,z)}{kT}}$$
- 分子总是优先占据势能较小的地方

**完整的 Boltzmann 分布（速度+位置）：**
$$dN = C \cdot e^{-\frac{\varepsilon_k + \varepsilon_p}{kT}} dv_x dv_y dv_z dx dy dz$$

核心结论：在温度 T 的平衡态下，粒子按状态的分布与 e^(-E/kT) 成正比。

### 10. 能量均分定理

**定理内容：** 在温度为 T 的热平衡态下，物质分子的**每个自由度都具有相同的平均动能 (1/2)kT**。

**注意：** 振动自由度除了动能 (1/2)kT 外，还有振动势能 (1/2)kT，合计每个振动自由度分配 kT。

**分子平均能量公式：** ε̄ = (1/2)(t + r + 2s) kT

| 分子类型 | t | r | s | 平均能量 | CV,m |
|----------|---|---|---|----------|------|
| 单原子分子 | 3 | 0 | 0 | (3/2)kT | (3/2)R |
| 刚性双原子分子 | 3 | 2 | 0 | (5/2)kT | (5/2)R |
| 非刚性双原子分子 | 3 | 2 | 1 | (7/2)kT | (7/2)R |
| 刚性多原子分子 | 3 | 3 | 0 | 3kT | 3R |

**自由度总结：**
- N 个原子组成的分子总自由度 = 3N (N ≥ 3)
- 平动自由度 = 3，转动自由度 = 3（非线性分子），振动自由度 = 3N - 6

### 11. 理想气体内能

$$\boxed{E = N\bar{\varepsilon} = \frac{1}{2}(t + r + 2s) NkT = \frac{1}{2}(t + r + 2s) \nu RT}$$

- 理想气体内能只是温度的函数，与体积无关

## 二、例题考点（第8章）

### 【例题1】氦气、氧气分子速率分布对比（P35-36）

**题目：** 氦气、氧气分子数均为 N，T_He = 2T_O₂，速率分布曲线如图，阴影面积为 S。求：(1) 哪条是氦气？(2) 对应曲线；(3) v=v₀时物理意义；(4) 阴影面积物理意义。

**解：**
- 由 vp = √(2RT/M)：vp,He / vp,O₂ = √(T_He·M_O₂ / T_O₂·M_He) = √(2×32 / 1×4) = 4
- 因此 O₂ 的曲线是 B（vp 较小的为氧气），氦气是 A
- 在 v=v₀ 时两种气体分子分布几率密度相同
- 阴影面积 = N ∫[fA(v)-fB(v)]dv 表示两种气体分子速率大于 v₀ 的分子数之差，数值为 (1-S)N

**考点：** 最概然速率公式，分布函数比较，物理意义分析。

### 【例题2】两种气体混合的速率分布函数（P37）

**题目：** 设 A、B 两种理想气体分子数分别为 NA、NB，某温度下各自速率分布函数为 fA(v)、fB(v)，求混合系统的速率分布函数。

**解：** 按定义 f(v) = dN/(N dv)
- 在 [v, v+dv] 内：dN = dNA + dNB = NA fA dv + NB fB(v) dv
- 总分子数 N = NA + NB
- 混合后：**f(v) = [NA fA(v) + NB fB(v)] / (NA + NB)**

**考点：** 速率分布函数的定义和按权重叠加原理。

### 【例题3】电子气速率分布（P38）

**题目：** 假设有 N 个电子的电子气，速率分布函数 f(v) 与 v 关系如图（分段线性函数，f(0)=0, f(v₀)=a, f(2v₀)=0）。求：(1) a 的大小；(2) [0,v₀] 间粒子平均速率；(3) 全部粒子平均速率。

**解：**
- 由归一化条件 ∫₀^{2v₀} f(v)dv = 1 → (1/2)·(2v₀)·a = 1 → **a = 1/v₀**
- [0,v₀] 区间平均速率：**2v₀/3**
- 全部粒子平均速率：**v₀**

**考点：** 归一化条件，分段函数的积分求平均速率。

### 【例题4】容器突然停止的温度变化（P70-71）

**题目：** 容器内储有质量为 m、摩尔质量为 M 的理想气体，容器以速度 V 定向运动，今使容器突然停止。求系统温度改变和分子速度平方平均值的增量：(1) 单原子分子；(2) 双原子刚性分子。

**解：** 定向运动动能 (1/2)mV² 全部转化为内能。

对于 (1) 单原子分子：E = ν·(3/2)RT
- ΔE = ν·(3/2)R·ΔT = (1/2)mV²
- **ΔT = MV²/(3R)**

对于 (2) 刚性双原子分子：E = ν·(5/2)RT
- **ΔT = MV²/(5R)**

分子速度平方平均值增量：**Δ(v²平均) = V²**

**考点：** 能量守恒，内能公式，能量均分定理在不同分子类型中的应用。

## 三、常见易错点（第8章）

1. **平衡态 ≠ 静止态：** 平衡态下分子仍在运动（动态平衡），只是宏观量不变。
2. **涨落是统计规律固有特性：** 分子数越少涨落越大；布朗运动是可观测的涨落现象。
3. **vp < v̄ < √(v²平均)，不要混淆三种速率的物理意义。**
4. **f(v) 是几率密度不是分子数；f(v)dv 才是比率。**
5. **温度是统计量：** 对单个分子谈温度无意义。
6. **相同温度下不同气体方均根速率不同：** 因摩尔质量不同。
7. **振动自由度分配 kT（不是 kT/2）：** 动能 kT/2 + 势能 kT/2。
8. **Boltzmann 分布：** 分子优先占据势能较小的地方。
9. **理想气体内能只与温度有关，与体积无关。**
10. **低温下某些自由度的"冻结"需量子力学解释。**

# 第二部分：第9章 热力学定律

---

## 一、核心公式/概念

### 1. 热力学基本概念

| 概念 | 要点 |
|------|------|
| 准静态过程 | 过程中每一状态都是平衡态，可在 p-V 图上用曲线表示 |
| 弛豫时间 | 非平衡态到平衡态的过渡时间，约 10⁻³ 秒 |
| 内能（理想气体） | E = (i/2)νRT，只与温度有关，是**状态量** |
| 功 | **过程量**，dA = p dV（系统对外作功）|
| 热量 | **过程量**，dQ > 0 吸热，dQ < 0 放热 |

**过程方程 vs 状态方程的区别：** 状态方程描述某状态的 P,V,T 关系；过程方程描述某过程中的变化规律。

### 2. 热力学第一定律

$$\boxed{dQ = dE + dA}$$

- $dA = p dV$（系统对外界作功）
- 各量均为代数量
- **另一种形式：** Q = ΔE + A

### 3. 等值过程汇总

| 过程 | 特征 | ΔE | Q | A（系统对外作功） |
|------|------|-----|----|----|
| 等容 | V=const | νCV,mΔT | νCV,mΔT | 0 |
| 等温 | T=const | 0 | νRT ln(V₂/V₁) | νRT ln(V₂/V₁) |
| 等压 | p=const | νCV,mΔT | νCp,mΔT | νRΔT |
| 绝热 | Q=0 | νCV,mΔT | 0 | -νCV,mΔT |

### 4. 热容量

**定容摩尔热容量：** CV,m = (i/2)R

**定压摩尔热容量（迈耶公式）：** Cp,m = CV,m + R

| 分子类型 | CV,m | Cp,m | γ = Cp,m/CV,m |
|----------|------|------|----------------|
| 单原子 (i=3) | (3/2)R | (5/2)R | 5/3 |
| 刚性双原子 (i=5) | (5/2)R | (7/2)R | 7/5 |
| 刚性多原子 (i=6) | 3R | 4R | 4/3 |

**热容比公式：** γ = 1 + 2/i

### 5. 绝热过程

**绝热自由膨胀：** Q=0, A=0 → ΔE=0 → 理想气体温度不变
- **注意：** 绝热自由膨胀 ≠ 准静态绝热过程，pV^γ = C 不适用！

**准静态绝热过程方程：**
$$\boxed{pV^\gamma = C_1} \quad \boxed{TV^{\gamma-1} = C_2} \quad \boxed{p^{\gamma-1}T^{-\gamma} = C_3}$$

**绝热线比等温线更陡：** 绝热过程中压强降低来自体积膨胀 + 温度降低两个方面。

### 6. 循环过程

**特征：** 系统回到初始状态 → ΔE = 0

**正循环（热机）：**
- 净功 A = Q₁ - |Q₂|（循环曲线包围的面积）
- **效率：** $$\eta = \frac{A}{Q_1} = 1 - \frac{|Q_2|}{Q_1}$$

**逆循环（致冷机）：**
- **致冷系数：** $$w = \frac{Q_2}{A} = \frac{Q_2}{|Q_1| - Q_2}$$

### 7. 卡诺循环与卡诺定理

**卡诺循环：** 两个等温过程 + 两个绝热过程

**卡诺热机效率：**
$$\boxed{\eta_{\text{卡诺}} = 1 - \frac{T_2}{T_1}}$$

**提高效率途径：** 提高 T₁，降低 T₂。

**卡诺致冷系数：** w_卡诺 = T₂/(T₁-T₂)

**卡诺定理：**
1. 在相同高低温热源之间工作的**一切可逆热机效率相等**，与工作物质无关：η = 1 - T₂/T₁
2. 一切**不可逆热机的效率小于可逆热机**：η < 1 - T₂/T₁

### 8. Otto循环（汽油机理想循环）

- ab：绝热压缩 → bc：等容吸热 → cd：绝热膨胀 → da：等容放热

**效率：** η = 1 - 1/r^(γ-1)，其中 r = V₁/V₂（压缩比）

- 汽油机 r < 11，理想效率 ~80%，实际 ~30%
- 柴油机实际效率 ~40%

### 9. 热力学第二定律

**两种经典表述（完全等价）：**

| 表述 | 内容 | 本质 |
|------|------|------|
| 克劳修斯 (1850) | 热量不能自动地由低温物体传向高温物体 | 热传导不可逆 |
| 开尔文-普朗克 (1851) | 不可能从单一热源吸收热量使之全部变成有用功而不产生其他影响 | 功热转换不可逆 |

**推论：** 单热源热机不可能制成 → **第二类永动机不存在！**

### 10. 可逆过程与不可逆过程

**可逆过程条件：** 系统回到初态 + **外界也完全复原**。

**不可逆性的两大因素：**
1. 不平衡因素（力学、热学、密度、温度的不平衡）
2. 耗散因素（摩擦等涉及功热转换）

**核心结论：** 无耗散的准静态过程是理想的可逆过程；一切与热现象有关的实际宏观过程都是不可逆的；所有宏观过程的不可逆性相互等价。

### 11. 热力学第二定律的统计意义

**微观解释：**
- 功变热：有序运动 → 无序运动
- 热传导：分子动能可区分 → 不可区分，无序性增大
- 气体绝热自由膨胀：小空间 → 大空间，位置分布更无序

**核心结论：** 一切自然过程总是沿着**无序性增大**的方向进行。

- 热力学第二定律是统计规律（只对大量粒子系统成立）
- 对应微观状态数目多的宏观状态出现的几率最大
- 孤立系统从热力学几率小的状态向几率大的状态进行
- 平衡态是对应微观状态数目最多的状态（几率最大）

**气体自由膨胀统计分析：**
- N 个分子全部返回 A 的几率为 (1/2)^N → N = 10²³ 时 → 0
- 两侧粒子数基本相同时对应平衡态，几率接近 1
- 宏观态热力学几率 Ω = N!/(n!(N-n)!)，n = N/2 时 Ω 最大

## 二、例题考点（第9章）

### 【例题1】绝热气缸中两部分气体（P24）

**题目：** 绝热气缸中有一固定的导热板 C，把气缸分为 A、B 两部分，D 是绝热活塞。A、B 分别盛有 1mol 氦气（He）和氮气（N₂）。若活塞慢慢压缩 A，所做功为 A，求：(1) B 部分内能变化；(2) A 部分摩尔热容。

**解：**
- 导热板 C → A、B 温度始终相等
- 绝热气缸 → QA + QB = 0
- He：CV,A = (3/2)R；N₂：CV,B = (5/2)R

(1) A = ΔEA + ΔEB = [(3/2)+(5/2)]RΔT = 4RΔT
- **ΔT = A/(4R)**
- **ΔEB = (5/2)R·A/(4R) = (5/8)A**

(2) CA = ΔQA/ΔT = -ΔQB/ΔT = -(-ΔEB)/ΔT = -(5/2)R
- **CA 为负值**（A 部分实际是从 B 部分吸热）

**考点：** 热力学第一定律、能量守恒、导热板保证温度相等条件、不同分子类型的内能公式。

### 【例题2】循环过程效率计算（P38）

**题目：** 气缸内有一定量氧气，经历循环过程：ab 等温、bc 等容、ca 绝热。已知 a 点 pa, Va，b 点 Vb = 3Va，求循环效率。

**解：**
- ab 等温吸热：Q₁ = pa Va ln 3
- bc 等容放热：Q₂ = (m/M) CV,m(Tc-Tb)，Tb = Ta
- ca 绝热过程：Ta Va^(γ-1) = Tc (3Va)^(γ-1) → Tc = Ta·3^(1-γ)
- O₂ 为刚性双原子，γ = 7/5 → Tc = Ta·3^(-2/5)
- **η = Q₁ - |Q₂| / Q₁ ≈ 19%**

**考点：** 循环效率公式、等温热量、绝热过程方程、等容热量。

### 【例题3】证明 p-V 图上两绝热线不相交（P48）

**题目：** 试证在 p-V 图上两绝热线不相交。

**证（反证法）：** 假设两绝热线交于 A 点。作一等温线与两绝热线分别交于 B、C。形成循环 BCAB。在此循环中，等温过程从单一热源吸收热量并全部转变为功，违反热力学第二定律（开尔文表述），所以假设错误。

**结论：p-V 图上两绝热线不相交！**

**考点：** 热力学第二定律的应用、反证法。

### 【例题4】核电厂效率计算（P56）

**题目：** 核电厂核反应热功率 1590MW，输电功率 540MW，水蒸气 556K，冷凝器 313K。求理想和实际效率。

**解：**
- 理想效率（卡诺）：η_ideal = 1 - 313/556 ≈ 43.7%
- 实际效率：η_actual = 540/1590 ≈ 34.0%
- 排入河水热量：1590 - 540 = 1050MW

**考点：** 卡诺效率公式、实际效率计算、能量守恒。

### 【例题5】气体向真空绝热自由膨胀分析（P9/23）

**题目：** 气体向真空绝热自由膨胀，抽去隔板前后水温不变。问：气体吸热？温度？内能？作功？

**分析：**
- Q = 0（绝热），A = 0（向真空膨胀无阻力）
- 热力学第一定律：ΔE = 0
- 理想气体 ΔE = 0 → ΔT = 0

**结论：气体温度不变！**

**注意：** 绝热自由膨胀 ≠ 准静态绝热过程（pV^γ = C 不适用）。初末态：V₂ = 2V₁ 时 p₂ = p₁/2（理想气体等温）。

**考点：** 热力学第一定律、自由膨胀的特殊性（非准静态）。

## 三、常见易错点（第9章）

1. **状态量 vs 过程量：** 内能 E 是状态量（只与初末态有关），热量 Q 和功 A 是过程量（与过程路径有关）。

2. **功的符号约定：** 在 dQ = dE + dA 中，dA 是系统**对外界**作的功。外界对系统作功时 dA < 0。

3. **绝热自由膨胀 pV^γ = C 不适用：** 只适用于准静态绝热过程。绝热自由膨胀初末态温度相同。

4. **热容量是过程量：** 不同过程有不同的热容量。

5. **卡诺效率公式中的温度必须是热力学温标（开尔文）。**

6. **第一类永动机 vs 第二类永动机：** 前者违反热一定律（η>1），后者违反热二定律（η=1 但从单一热源取热）。

7. **可逆过程要求系统 + 外界都复原：** 仅有逆过程回到初态不算可逆，必须外界也复原。

8. **所有宏观不可逆过程相互等价：** 假设其中一个可逆会推导出另一个也可逆。

9. **热力学第二定律是统计规律：** 只适用于大量粒子系统，对少数粒子不成立。

10. **Otto 循环效率 η = 1 - 1/r^(γ-1)：** γ 是绝热指数，r 是压缩比。
